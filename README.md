
# Waschbär Engine

  

Waschbär Engine is a multithreaded 2D game engine with a Unity-style actor/component architecture with Lua scripting.

  

## Installation

Currently this is built across Make, Visual Studio, and XCode for cross-platform support but I'm looking to migrate to CMake soon.

Using Makefile:

```bash
cd  path/to/project
make
./waschbaer_engine
```
